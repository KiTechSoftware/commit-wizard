# Commit Wizard Documentation

Commit Wizard (`cw`) is a lightweight CLI tool that helps teams enforce **Conventional Commits**, manage **Semantic Versioning**, and automate **changelog generation**.

Commit Wizard should feel like “a spellbook for your commits” — simple recipes, powerful results.

It guides developers like an experienced mentor, ensures consistency for teams, and gives CI/CD pipelines strict, reproducible enforcement.

<img src="./assets/img/mascot.png" alt="Commit Wizard Mascot" width="200" align="right" />

## 🔑 Design Philosophy

Commit Wizard is built around a few core principles:

- **Guided, not restrictive** → help developers without blocking them
- **Simple mental model** → one config at a time, explicit overrides
- **Portable & reliable** → works the same on Linux, macOS, Windows, Docker/Podman
- **CI-friendly** → interactive for devs, strict for automation
- **[Conventional Commits first](https://www.conventionalcommits.org)** → `feat` and `fix` always enforced by default; teams can extend
